Trigger = Caps_lock

Trigger + J/I/K/L            = Arrow Keys
Trigger + U/O                = d9
Trigger + Spacebar           = Fn
Trigger + Spacebar + J/I/K/L = Home / Page Down / Page Up / End
Trigger + Spacebar + U/O     = Go to Start/End of Line

Trigger + A + J/I/K/L            = Select Text
Trigger + A + U/O                = Select until Start/End of Next Word
Trigger + A + 9/0                = Select until Start/End of Line
Trigger + A + Open/Close Bracket = Select until Start/End the Paragraph

Trigger + H = Forward Delete
Trigger + N = Delete

Trigger + S = Cut
Trigger + D = Copy
Trigger + F = Paste

Trigger + R = Undo
Trigger + T = Redo

Trigger + M      = Find
Trigger + Comma  = Find Previous
Trigger + Period = Find Next
